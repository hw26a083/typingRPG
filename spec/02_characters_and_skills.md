# タイピングRPG仕様書 - 02. キャラクターとレベルアップシステム

このドキュメントは、プレイヤーが操作するパーティーキャラクター、レベルアップによる成長、および修得するワザ（スキル）について定義する。

---

## 1. パーティーキャラクター構成
プレイヤーは以下の4人パーティーを操作する。戦闘時の行動選択およびタイピングは、以下の並び順（素早さ順）で固定される。

### 1.1 勇者 (Hero)
* **設定**: 男の子。剣を扱う万能タイプ。
* **役割**: 近接物理攻撃、バランスの良いアタッカー。
* **行動順**: 1番手

### 1.2 魔法使い (Mage)
* **設定**: 女性。強力な魔法を操る。
* **役割**: 属性魔法による単体・全体への大ダメージ。
* **行動順**: 2番手

### 1.3 僧侶 (Priest)
* **設定**: 老人の男性。神聖な祈りで味方を癒す。
* **役割**: 回復、防御強化、生存補助。
* **行動順**: 3番手

### 1.4 戦士 (Warrior)
* **設定**: 屈強な男性。強大な力で一撃を加える。
* **役割**: 高火力の物理攻撃、高い耐久力。
* **行動順**: 4番手

---

## 2. レベルアップと経験値システム
モンスターを撃破すると、戦闘終了時に経験値を獲得できる。

* **経験値の獲得**: 倒したモンスターごとに設定された経験値がパーティー全体、または生存メンバーに配分される。
* **レベルアップ**: 累積経験値が一定値に達するとキャラクターのレベルが上昇する。
  * レベルアップ時、最大HPなどのステータスが上昇する（具体的な数値は[要検討]）。
  * レベルアップすると、キャラクター固有の**「新たなワザ（技）」**を修得する。

---

## 3. ワザ（スキル）とタイピングワード
バトル時に各キャラクターが使用できるワザには、それぞれ「ワザ名」と「タイピングするワード（ローマ字）」が設定されている。

### 3.1 勇者のワザ候補（例）
| ワザ名 | タイピングワード | 効果 | 習得条件 |
| :--- | :--- | :--- | :--- |
| 斬る | `kiru` | 単体物理ダメージ（小） | 初期ワザ |
| なぎ払い | `nagiharai` | 全体物理ダメージ（中） | レベル2 |
| ※その他のワザ | [要検討] | | |

| :--- | :--- | :--- | :--- |
| 火の玉 | `hinotama` | 単体魔法ダメージ（中） | 初期ワザ |
| ※その他のワザ | [要検討] | | |
