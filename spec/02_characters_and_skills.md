# タイピングRPG仕様書 - 02. キャラクターとレベルアップシステム

このドキュメントは、プレイヤーが操作するパーティーキャラクター、レベルアップによる成長、および修得するワザ（スキル）について定義する。

---

## 1. パーティーキャラクター構成
プレイヤーは以下の4人パーティーを操作する。戦闘時の行動選択およびタイピングは、以下の並び順（素早さ順）で固定される。

### 1.1 勇者 (Hero)
* **設定**: 男の子。剣を扱う万能タイプ。
* **役割**: 近接物理攻撃、バランスの良いアタッカー。
* **行動順**: 1番手

### 1.2 魔法使い (Mage)
* **設定**: 女性。強力な魔法を操る。
* **役割**: 属性魔法による単体・全体への大ダメージ。
* **行動順**: 2番手

### 1.3 僧侶 (Priest)
* **設定**: 老人の男性。神聖な祈りで味方を癒す。
* **役割**: 回復、防御強化、生存補助。
* **行動順**: 3番手

### 1.4 戦士 (Warrior)
* **設定**: 屈強な男性。強大な力で一撃を加える。
* **役割**: 高火力の物理攻撃、高い耐久力。
* **行動順**: 4番手

---

## 2. レベルアップと経験値システム
モンスターを倒すことで経験値を獲得し、キャラクターが成長する。

* **経験値の獲得**: 戦闘でモンスターを倒すと、経験値を入手できる。
* **レベルアップの仕組み**:
  * ステージをクリアし、**次のステージへ移動したタイミング**でパーティーメンバー全員のレベルが上昇する。
  * ステージごとのキャラクターレベルは以下の通りとなり、最終ステージではすべてのワザが習得された状態になる。
    * **ステージ1 (草原)**: レベル1 (各キャラクターの1番目のワザのみ使用可能)
    * **ステージ2 (砂漠)**: レベル2 (2番目のワザを習得)
    * **ステージ3 (雪原)**: レベル3 (3番目のワザを習得)
    * **ステージ4 (火山)**: レベル4 (最終奥義を含むすべてのワザを習得)
  * レベルアップ時、最大HPなどのステータスが上昇する（具体的な数値は[要検討]）。

## 3. ワザ（スキル）とタイピングワード
バトル時に各キャラクターが使用できるワザの一覧。各ワザには「ワザ名」「タイピングワード」「効果」が設定されている。

※各ワザの習得レベルについては[要検討・質問事項]。

| :--- | :--- | :--- | :--- |
| 回転切り | `kaitengiri` | 選択した敵とその左右にいる敵に10ダメージ与える。 | レベル1 (初期ワザ) |
| 炎舞切り | `enbugiri` | 選択した敵に20ダメージを与える。10%の確率でやけど状態にし、3ダメージずつ削る（ランダムなターン数継続）。 | レベル2 |
| 氷雪切り | `hyousetugiri` | 選択した敵に30ダメージを与える。10%の確率でこおり状態にし、その敵は何もすることができなくなる（ランダムなターン数継続）。 | レベル3 |
