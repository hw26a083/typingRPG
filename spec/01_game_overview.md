# Typing Quest 仕様書 - 01. ゲーム概要

このドキュメントは、タイピングを入力して戦闘を攻略するRPGゲーム「Typing Quest」の全体概要を定義した仕様書である。

---

## 1. 基本情報
* **ゲームタイトル**: Typing Quest
* **ゲームジャンル**: タイピングRPG
* **グラフィックスタイル**: 2Dレトロドット絵スタイル
* **対象プラットフォーム**: Webブラウザ / PC（キーボード操作推奨）
* **操作方法**: 
  * **完全キーボード操作集約**:
    * 移動（十字キー）、コマンド選択・ターゲット選択（矢印キー/WASDでのカーソル移動、Enter/Spaceでの決定）、およびタイピング攻撃（アルファベット入力）のすべてがキーボード操作に集約されている。マウスなしで全編をシームレスにプレイ可能。
* **画面レイアウト（スクロール制御）**:
  * プレイヤーの没入感とタイピングの集中を妨げないため、**ゲーム画面は常にビューポート（1画面）に収まり、縦方向・横方向のスクロールバーは一切発生しない仕様（`overflow: hidden`等による完全スクロール排除）**とする。戦闘中、およびリザルト表示時も画面サイズ内に完全固定される。

---

## 2. 世界観とストーリー
プレイヤーは男の子の勇者となり、仲間である魔法使い、僧侶、戦士と共に、世界を脅かす魔王を倒すための冒険に出る。5つの過酷な環境のステージを乗り越え、最終ステージに待ち受ける魔王の討伐を目指す。

---

## 3. ステージ構成
ゲームは全5ステージで構成され、ステージが進むごとに登場するモンスターが強力になる。

| ステージ | テーマ | 特徴 | 登場モンスター数 |
| :--- | :--- | :--- | :--- |
| **ステージ1** | 草原 | 冒険の始まり。基本的なタイピング練習を兼ねた優しいステージ。 | 3匹 |
| **ステージ2** | 砂漠 | 乾いた荒野。少しタイピング文字数が長く、速度が求められる。 | 3匹 |
| **ステージ3** | 雪原 | 凍てつく氷の世界。モンスターの攻撃が強力になり、素やすい防御が必要。 | 3匹 |
| **ステージ4** | 火山 | 灼熱の溶岩地帯。極限環境のモンスターが立ち塞がる。 | 3匹 |
| **ステージ5** | 魔王城 | 最終決戦の地。ボスの魔王のみが君臨している。 | 魔王1体のみ |

### 3.1 ステージの遷移と回復
* 各ステージのエリアを攻略し、次のステージへ進む（ポータルに入る）際、**味方キャラクター全員の最大HP上限が自動的に +20 される**。
* 最大HPの上限アップと同時に、**全員の体力が全回復**する。
* 前のステージで**死亡（戦闘不能）していたキャラクターも自動的に復活**し、体力が最大まで回復した状態で次のステージを開始する。

---

## 4. 移動システム
* **基本操作**: 十字キーを使用して、ステージ内を自由に歩き回ることができる。
* **エンカウント方式**: シンボルエンカウント
* **現在のステージ表示**: 移動画面上部に現在のステージ名を表示する（例：「草原」など）。

---

## 5. テスト機能（テストモード）
* **機能概要**: デバッグやゲーム検証用のテストモードを搭載する。
* **トリガー**: 画面右上の音声ON/OFFボタンを**キーボードのKキーを押しながらクリック**すると、テストモードのON/OFFが切り替わる。
* **テストモード中の特別仕様**:
  * **ステージ選択（ジャンプ）**: 画面上部にステージ1〜5の選択ボタン一覧が表示され、クリックすることで任意のステージを自由に行き来できる。ステージ遷移時は味方パーティー全員が全回復する。
  * **スキルの全解放**: 勇者パーティー全員のすべての技（レベル制限のあるスキルや最終奥義）がレベルに関わらず全解放され、いつでも選択可能になる。
  * **切り戻し**: 再度Kキーを押しながら右上ボタンをクリックするとテストモードが終了し、通常の状態（レベルに基づいたスキル制限やステージ移動制限）に綺麗に戻る。
