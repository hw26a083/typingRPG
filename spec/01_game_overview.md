# タイピングRPG仕様書 - 01. ゲーム概要

このドキュメントは、タイピングを入力して戦闘を攻略するRPGゲームの全体概要を定義した仕様書である。

---

## 1. 基本情報
* **ゲームジャンル**: タイピングRPG
* **グラフィックスタイル**: 2Dレトロドット絵スタイル
* **対象プラットフォーム**: Webブラウザ / PC（キーボード操作推奨）
* **操作方法**: 
  * 移動・選択: キーボード（詳細は要検討）
  * バトル時: アルファベットキーボード入力（ローマ字入力）

---

## 2. 世界観とストーリー
プレイヤーは男の子の勇者となり、仲間である魔法使い、僧侶、戦士と共に、世界を脅かす魔王を倒すための冒険に出る。4つの過酷な環境のステージを乗り越え、最終ステージに待ち受ける魔王の討伐を目指す。

---

## 3. ステージ構成
ゲームは全4ステージで構成され、ステージが進むごとに登場するモンスターが強力になる。

| ステージ | テーマ | 特徴 | 登場モンスター数 |
| :--- | :--- | :--- | :--- |
| **ステージ1** | 草原 | 冒険の始まり。基本的なタイピング練習を兼ねた優しいステージ。 | 2匹以上 |
| **ステージ2** | 砂漠 | 乾いた荒野。少しタイピング文字数が長く、速度が求められる。 | 2匹以上 |
| **ステージ3** | 雪原 | 凍てつく氷の世界。モンスターの攻撃が強力になり、素早い防御が必要。 | 2匹以上 |
| **ステージ4** | 火山 | 最終決戦の地。最奥にはボスの魔王が待ち受けている。 | 2匹以上 + 魔王 |

### 3.1 ステージの遷移と回復
* 各ステージのエリアを攻略し、次のステージへ進む際、**味方キャラクター全員の体力が全回復**する。
* 前のステージで**死亡（戦闘不能）していたキャラクターも自動的に復活**し、体力が最大まで回復した状態で次のステージを開始する。

---

## 4. 移動システム
* **エンカウント方式**: ランダムエンカウント
  * ステージ内を歩いていると、一定確率で突然モンスターと遭遇し、シームレスにバトル画面に切り替わる。
